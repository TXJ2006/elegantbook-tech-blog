\chapter{为什么用 LaTeX 写技术博客}

LaTeX 很适合写包含公式、图表、算法、引用和长篇结构的技术文章。这个工程把每篇文章写成一个独立的 \verb|.tex| 文件，并在发布时自动生成 PDF 与网页预览。

\section{公式排版}

例如一个常见的优化目标可以写成：
\[
  \min_{\theta} \frac{1}{n}\sum_{i=1}^{n}\ell(f_{\theta}(x_i), y_i) + \lambda\|\theta\|_2^2.
\]

\section{代码高亮}

下面的代码块使用 \verb|listings| 和 \verb|tcolorbox| 渲染，不需要额外开启 \verb|minted| 的 shell escape，也适合 GitHub Actions 自动编译。

\begin{codeblock}{Python}
from dataclasses import dataclass

@dataclass
class Experiment:
    name: str
    learning_rate: float = 3e-4

def train(config: Experiment) -> None:
    print(f"running {config.name} at lr={config.learning_rate}")

train(Experiment("latex-blog"))
\end{codeblock}
